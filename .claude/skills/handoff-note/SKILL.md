# Handoff Note

Create or update `continue.md` with:

1. Objective and current status.
2. Files changed or created.
3. Verification commands run and results.
4. Remaining tasks in priority order.
5. Known blockers, assumptions, and first files to inspect.

Keep the note factual. Do not include long chat history or speculation.
